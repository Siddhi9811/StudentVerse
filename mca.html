Master of Computer Applications (MCA)

The Master of Computer Applications (MCA) is a three-year postgraduate program that focuses on developing advanced skills in computer applications, software development, and IT management.

Curriculum

Semester Subjects
Semester 1 Mathematics for Computing, Computer Programming, Computer Organization, Data Structures, Communication Skills
Semester 2 Object-Oriented Programming, Database Management Systems, Software Engineering, Operating Systems, Elective I
Semester 3 Web Technologies, Mobile Application Development, Computer Networks, Elective II, Project Work (Part I)
Semester 4 Advanced Java Programming, Cloud Computing, Data Mining, Elective III, Project Work (Part II)
Semester 5 Machine Learning, Cyber Security, Big Data Analytics, Elective IV, Major Project
Semester 6 Software Testing, IT Project Management, Elective V, Internship, Major Project Presentation
